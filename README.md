# clip-star

A Raycast extension for bookmark management, supporting quick identification of link content, generation of titles and tags, as well as creation of link screenshots.

## Features

- Quickly identify link content, generate titles and tags
  - Supports recognition of page titles and descriptions
  - Also supports AI-generated titles and descriptions (requires your own OpenAI key)
- Generate link screenshots
- Supports local export and saving of data, as well as integration with Notion database for content synchronization
- Supports Chinese and English
